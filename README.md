# 35-space-rookie
35 y.o. Space-Rookie

Интерактивный симулятор посадки космического аппарата на поверхность Европы (спутник Юпитера) с научно достоверной физической моделью.

📋 Описание
Симулятор моделирует посадку космического аппарата на Европу с учётом уникальных условий спутника:
Низкая гравитация — 1.314 м/с²
Отсутствие атмосферы — только реактивная посадка
Высокий радиационный фон — накопление дозы, отказ электроники
Экстремальные температуры — –160°C на поверхности
Сложный ледяной рельеф — трещины, хребты, уклоны
Пользователь управляет тягой, ориентацией и радиационной защитой, балансируя между топливом, радиацией и температурой.

🎮 Управление
Клавиша	Действие
↑ / ↓	Увеличение / уменьшение тяги двигателя
← / →	Наклон аппарата влево / вправо
R	Включение / выключение радиационной защиты
T	Выбор цели посадки (после нажатия кликнуть по поверхности)
Пробел	Перезапуск симуляции
P	Пауза
Shift	Ускорение времени (x10)
Мышь	Клик по поверхности — выбор точки посадки

Текстовое описание архитектуры
Проект состоит из 4 основных классов:

1. Physics — физические константы
Хранит все научно обоснованные параметры Европы и аппарата:

Гравитация, радиус планеты

Характеристики двигателя (тяга, удельный импульс)

Параметры радиации (доза на поверхности/орбите, порог отказа)

Температурные пороги

Посадочные критерии (скорость, уклон)

Начальные условия

2. Terrain — рельеф поверхности
Процедурная генерация ледяной поверхности с трещинами и хребтами

Метод get_height(x) — возвращает высоту рельефа в заданной точке

Метод get_slope(x) — возвращает уклон в градусах

3. Lander — физическая модель аппарата
Состояние: позиция, скорость, масса, топливо

Управление: тяга (0–1), угол наклона, радиационная защита

Метод update(dt, terrain) — обновление состояния через RK4

Метод get_radiation_rate(altitude) — расчёт мощности дозы

Метод check_landing(terrain) — проверка условий посадки

4. Visualizer — визуализация (Pygame)
Отрисовка поверхности, аппарата, струи двигателя

Отрисовка телеметрии и панели управления

Следование камеры за аппаратом

Автоматическое масштабирование при снижении
